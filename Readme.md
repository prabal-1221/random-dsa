# Useful uv Commands
| Task               | Command                |
| ------------------ | ---------------------- |
| Create project     | `uv init`              |
| Create virtual env | `uv venv`              |
| Install dep        | `uv add pkg`           |
| Install dev dep    | `uv add --dev pkg`     |
| Remove dep         | `uv remove pkg`        |
| Run script         | `uv run script.py`     |
| Lock & sync deps   | `uv sync`              |
| List installed     | `uv pip list`          |
| Upgrade dependency | `uv add --upgrade pkg` |
